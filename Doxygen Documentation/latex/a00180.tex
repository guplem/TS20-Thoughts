Tries to execute the next planned even in the execution\+Plans list. 

If the distance to execute the \mbox{\hyperlink{a00172}{Map\+Event}} is not met, an order to move this \mbox{\hyperlink{a00180}{Map\+Element}} is given instead.

After executing the next planned event, an order to execute the next one is going to be given immediately.

It is considered the \char`\"{}next\char`\"{} map event the last one added in the list.

\begin{DoxyReturn}{Returns}
True if the behaviour was expected (the planned event was executed successfully, the distance was not met so the object was moved, ...). False if the planned event could not be executed due to unexpected reasons.
\end{DoxyReturn}
