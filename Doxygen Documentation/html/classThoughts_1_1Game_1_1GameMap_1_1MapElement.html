◆ DoNextPlanedMapEvents()

bool Thoughts.Game.GameMap.MapElement.DoNextPlanedMapEvents ( )
private

Tries to execute the next planned even in the executionPlans list.

If the distance to execute the MapEvent is not met, an order to move this MapElement is given instead.

After executing the next planned event, an order to execute the next one is going to be given immediately.

It is considered the "next" map event the last one added in the list.

Returns
True if the behaviour was expected (the planned event was executed successfully, the distance was not met so the object was moved, ...). False if the planned event could not be executed due to unexpected reasons.
